# BSM307 – Bilgisayar Ağları Dönem Projesi

## Proje Konusu
Bu projede çok amaçlı yönlendirme problemi ele alınmıştır.
Amaçlar:
- Toplam gecikmenin minimize edilmesi
- Yol güvenilirliğinin maksimize edilmesi
- Bant genişliği kullanım maliyetinin minimize edilmesi

## Kullanılan Algoritmalar
- Genetik Algoritma (GA)
- Simulated Annealing (SA)
- Dijkstra algoritması (karşılaştırma amacıyla)

## Ağ Modeli
- Düğüm sayısı: 250
- Ağ modeli: Erdős–Rényi (p = 0.4)

## Tekrarlanabilirlik
Deneylerde sabit seed değeri kullanılmıştır:
seed = 42
